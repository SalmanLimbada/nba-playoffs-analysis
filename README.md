# NBA Playoffs Analysis (Assignment 1)

This project is for my Scientific Data Analysis assignment.

I used the 2023-2024 NBA playoffs player stats dataset to study one main question:

How does player position relate to points scored in the playoffs?

## Files in this project

- `assignment 1.ipynb` - main notebook with the full analysis
- `2023-2024 NBA Player Stats - Playoffs.csv` - dataset used in the notebook

## What I did

- Loaded the CSV file into pandas
- Cleaned the data by dropping unneeded columns and renaming some columns
- Checked for missing values
- Compared average points by position (PG, SG, SF, PF, C)
- Created visualizations like bar plots, box plots, and scatter plots
- Looked at specific cases like Mavericks vs Celtics and top 10 scorers
- Discussed possible reasons why Point Guards scored the most on average

## Main result

From this dataset, Point Guards had the highest average points.
A possible reason is that they usually play more minutes and have a strong role in both scoring and playmaking.
